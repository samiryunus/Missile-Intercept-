# Missile Intercept Simulation (MATLAB)

## Overview
This project simulates a missile intercept scenario using proportional navigation
and visualizes the engagement in a 2D UI.

## Requirements
- MATLAB R2021a or newer
- No additional toolboxes required

## How to Run
1. Open MATLAB
2. Navigate to the project folder
3. Run:
   missile_intercept_ui_FINAL

## Controls
- Start / Reset buttons in UI
- Adjustable PN gain and initial conditions

## Notes
- Educational simulation (not classified / not real-world tuned)
